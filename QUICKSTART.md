# ⚡ Quick Start - Registro Lavoro PWA

## ✅ File pronti al download

La tua PWA è completa e pronta! Ecco cosa hai ricevuto:

### 📁 File principali:
- **`index.html`** ← L'app (apri questo file!)
- **`manifest.json`** ← Configurazione PWA
- **`service-worker.js`** ← Funzionamento offline
- **`README.md`** ← Documentazione completa
- **`GITHUB_DEPLOY.md`** ← Come metterla online

---

## 🚀 3 modi per usarla

### **1️⃣ Subito sul tuo PC (più semplice)**
1. Scarica la cartella `pwa-registro-lavoro`
2. Apri `index.html` con Chrome/Firefox
3. ✅ Fatto! L'app funziona

**Vantaggi:** Immediato, niente configurazione  
**Svantaggi:** Solo offline locale

---

### **2️⃣ Online gratis con GitHub (consigliato)**
1. Segui le istruzioni in `GITHUB_DEPLOY.md`
2. In 5 minuti sarà online
3. Installabile su mobile come app vera

**Vantaggi:** Online, installabile, HTTPS, gratuito  
**Svantaggi:** Serve account GitHub

---

### **3️⃣ Sul tuo server web**
1. Carica i file con FTP
2. Accedi via browser
3. ✅ PWA completa con HTTPS

**Vantaggi:** Pieno controllo, veloce  
**Svantaggi:** Serve un server

---

## 📱 Come installarla su mobile

### Android (Chrome):
```
1. Apri il sito da mobile
2. Clicca il popup "Installa"
3. L'app appare in home
```

### iPhone (Safari):
```
1. Apri il sito da Safari
2. Condividi > "Aggiungi a schermata home"
3. L'app appare in home
```

## 🔧 Personalizzazione (opzionale)

Vuoi cambiare nome/colore?

Modifica `manifest.json`:
```json
{
  "name": "Il tuo nome",
  "theme_color": "#FF0000"
}
```

Basta! Ricaricare la pagina.
